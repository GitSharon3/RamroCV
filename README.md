# Resume Builder

A modern, user-friendly web application for creating professional resumes. Built with React.js, Tailwind CSS, and Framer Motion.

**Live Demo:** https://gitsharon3.github.io/RamroCV

## Features

- **Dynamic Resume Builder** – Create and edit resumes with intuitive forms for personal info, education, work experience, skills, projects, and hobbies
- **Drag & Drop Sections** – Reorder resume sections seamlessly
- **Real-time Preview** – Live preview with multiple templates (Modern, Classic, Creative)
- **PDF Export** – Download resumes as professional PDFs
- **Responsive Design** – Fully optimized for mobile and desktop

## Tech Stack

React · Vite · Tailwind CSS · Zustand · Framer Motion · jsPDF

## Getting Started

```bash
npm install
npm run dev
```
